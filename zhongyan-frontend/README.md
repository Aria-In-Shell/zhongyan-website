# 江西中研官网前端原型

这是第一版静态前端原型，用于确认首页、产品中心、产品筛选、案例展示和询价路径。

## 打开方式

直接用浏览器打开 `index.html` 即可。

## 页面结构

- `index.html`：短首页，负责品牌定位、设备入口、主推设备、定制服务、案例摘要和询价入口。
- `products.html`：完整产品中心，包含产品筛选、产品卡片、详情弹窗和询价入口。

## 当前原则

- 产品数据先放在 `app.js` 中，后续可替换为 CMS 或后端接口。
- `生命周期状态`、`投产/上市日期`、`停产日期` 等字段只做内部管理，不在前台默认展示。
- 产品中心默认只展示 `visibleInCatalog: true` 的产品。
- 图片暂时复用原站公开图片链接，正式开发时需要统一压缩、改名、转 WebP/AVIF。
